Middleware Development for Automated File Scanning in OpenStack Swift (Python)

OpenStack is a free and open-source cloud computing platform, typically deployed as Infrastructure-as-a-Service (IaaS) in private and public cloud environments. A key component is OpenStack Swift, a distributed, highly available object storage service based on a microservice architecture. To enhance Swift's security, a custom Middleware was developed in Python. This Middleware intercepts uploaded files destined for storage and sends them to a Function-as-a-Service (FAAS) for security scanning by Multi-Antivirus (MultiAV) systems. If a file is flagged as malicious or problematic by the MultiAV system, it is moved to a quarantine area. Key system features:

  • Kubernetes-based deployment.
  • Microservice architecture.
  • Very high availability.
  • Utilizes FAAS (based on OpenWhisk).
  • Compatible with various domestic MultiAV systems like Bitbaan, Geraf, and Cyberno.
